SQL Joins - Complete Guide
📌 Overview
This repository covers the most commonly used SQL JOINs with practical examples and queries in PostgreSQL. It helps in understanding how data from multiple tables can be combined efficiently.
🚀 Topics Covered
1. INNER JOIN
Returns records that have matching values in both tables.
2. LEFT JOIN (LEFT OUTER JOIN)
Returns all records from the left table and matching records from the right table.
3. RIGHT JOIN (RIGHT OUTER JOIN)
Returns all records from the right table and matching records from the left table.
4. FULL JOIN (FULL OUTER JOIN)
Returns all records when there is a match in either table.
5. CROSS JOIN
Returns the Cartesian product of both tables.
6. SELF JOIN
Joins a table with itself to compare rows within the same table.
